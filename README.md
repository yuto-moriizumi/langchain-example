# LangChain example

Example Chat Q&A application with LangChain

## Installation

1. Clone this repository
1. `npm ci`
1. `npm run dev -w front` to launch the NextJS application

## Usage

### Post questions & get answers

1. Open http://localhost:3000
1. Post your question
